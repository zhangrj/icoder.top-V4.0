前端框架

一个前端开发框架其实就是一系列产品化的HTML/CSS/JavaScript组件的集合，框架可以让开发工作变得简单。利用框架，你可以花最少的力气创建响应式且符合标准的网站，使用框架有很多好处，比如说简单快速，以及在不同的设备之间的一致性等等。框架最大的优势就是简单易用，即使只掌握少量的web知识，你也可以毫无障碍的使用它们。

Bootstrap

Bootstrap是目前最受欢迎的HTML,CSS和,JS框架，由推特工程师开发，用动态语言LESS写成，用于开发响应式布局、移动设备优先的WEB项目，它的中文文档做的相当不错。

BootMetro

基于Bootstrap框架构建，用于创建Win8 Metro风格的网站，除了提供bootstrap的所有功能外，还包括页面平铺、应用程序栏等功能，用它构建的网站外观十分前卫。

Kickstrap

Kickstrap无缝整合了Bootstrap和顶级Web技术，并在它的基础上添加了许多app,主题以及附加功能，使用它无需一个本地后端即可运行一个数据库驱动的Web应用程序。

Fbootstrapp

fbootstrapp是一个用来快速开发Facebook的iframe的应用的工具包，它包括排版，表格，按钮，表格，网格，导航，以及基础CSS和HTML，提供典型的facebook风格样式。

Foundation

Foundation 是一个易用、强大而且灵活的响应式前端框架，用于构建基于任何设备上的响应式网站、Web应用和电子邮件。结构语义化、移动设备优先、完全可定制。

View details »

Groundwork

GroundworkCSS主要用于快速构建响应式 Web应用程序，拥有灵活的、可嵌套的网格系统，还提供多种 UI 组件，如导航、按钮、图标、表单、对话框、工具提示等等。

Gumby

Gumby是一个响应式CSS网格框架，支持 PC、平板和手机屏幕显示，类似Twitter的Bootstrap，它包括一个Web UI工具包，有好看按钮，表格，导航和标签。

HTML KickStart

HTML KickStart拥有极简的HTML5，CSS和JS构建模块，提供了多种样式表，以及一些跨浏览器兼容的web组件，能够快速生成网页，十分容易上手。

Ivory

这是一个灵活、强大的响应式Web框架，相比Bootstrap，Ivory是一个超轻量级的解决方案，提供12列的响应式网格布局，包含各种常见组件和样式

Kube

全球最为流行的CSS框架之一，如果你需要一个基础框架来搭建一个新的项目，又不希望很复杂，那么Kube刚好是个不错的选择，因为这个框架只有一个CSS文件。

Ink

Ink是一个全新的制作响应式EDM模板的框架，由Zurb团队开发，使用Ink框架创建的"EDM模板"可以在任何设备和客户端上运行，包括Outlook。

Maxmertkit

Skeleton

Skeleton是一个小的JS和CSS文件的集合,可帮你快速开发漂亮的网站,适合各种屏幕设备包括手机。Skeleton 基于 960 grid 开发，它是一个 UI 框架。

Helium

Helium是一个响应式Web框架，用于使用HTML5+CSS3快速制作原型以及开发产品，Helium 设计更加轻量级，而且更容易修改，Helium只有30k左右。

Markup

Markup是一套纯CSS框架,集布局、UI组件和字体排版样式为一体，可以帮助你快速的执行web相关的设计和开发，Markup框架不包括Javascript。

Topcoat

Top只由html+CSS构成，是一个快速响应式web框架，提供超过100个图标，使用Adobe Web字体，性能很优秀，并且可以通过简单的操作自定义主题。

purecss

Purecss是一组轻量级、响应式的CSS模块，Pure小的没有节操，全部模块gzip压缩后才4.4KB，Pure提供最基础的样式，鼓励你基于此写出自己的样式。

Boilerplate

HTML5 Boilerplate帮你构建快速,健壮,并且适应力强的web app或网站，这个小小的源码包集合了100位开发者的经验，你可以将这些经验运用在你的项目中。

UIKit

UIKit一款轻量级、模块化的前端框架，可快速构建强大的web前端界面，代码结构清晰简单，易于扩展和维护，拥有体积小、反应灵敏的响应式组件。

Montage

Montage是一款支持构建现代化Web应用程序的HTML5框架，能协助你开发出可扩展和可维护的HTML5应用，拥有可重用的组件和HTML模板，声明组件模型等功能。

Layers CSS

Layers CSS是一个除了专注于处理主要结构外，不着重处理任何其它设计的轻量级CSS框架。它支持响应式与流体网格布局，使用简单的类来处理它们。

52Framework

52Framework一个Web开发框架，支持HTML5和CSS3。它是一个跨浏览器的框架，可以在所有主流的浏览器上运行，包括IE6，它拥有大量优秀的组件。

CreateJS

CreateJS是基于HTML5开发的一套模块化的库和工具，拥有共同或独立工作的丰富交互式内容的开源Web技术,能够在所有现代桌面和移动浏览器正常运行.
