常用发行版下载

Linux 发行版一般为用户预先集成好Linux操作系统及各种应用软件，一般用户不需要重新编译，在直接安装之后，只需要小幅度更改设置就可以使用，通常以软件包管理系统来进行应用软件的管理。Linux发行版通常包含了包括桌面环境、办公包、媒体播放器、数据库等应用软件。现在有超过300个Linux发行版，大部分都正处于活跃的开发中，不断地改进。

由于大多数软件包是自由软件和开源软件，所以Linux发行版的形式多种多样——从功能齐全的桌面系统以及服务器系统到小型系统 (通常在嵌入式设备，或者启动软盘)。下面列出的是最常用的发行版，除了个别商业版，都可以免费下载使用。

Ubuntu

Ubuntu是当今最受欢迎的免费操作系统，作为Debian的一款衍生版，Ubuntu的进程、外观大多数仍然与Debian一样，使用apt软件管理工具来安装和更新软件，是最容易使用的发行版之一。

Debian

Debian运行起来极其稳定，这使得它非常适合用于服务器，由它派生出了多个Linux发行版，Debian这款操作系统无疑并不适合新手用户，而是适合系统管理员和高级用户，支持如今的大多数处理器架构。

View details »

Gentoo

Gentoo包含数量众多的软件包，安装和使用起来较为困难；不过它被认为是最佳学习对象，可以进而了解Linux操作系统的内部运作原理使用portage来安装和更新软件，适合对Linux驾轻就熟的用户。

View details »

RedHat

这是第一款面向商业市场的Linux发行版，它有服务器版本，支持众多处理器架构，红帽企业级Linux是系统管理员的第一选择，由于该发行版是商业化产品，所以不是免费的。教学版下载

View details »

CentOS

CentOS是一款企业级Linux发行版，它使用红帽企业级Linux中的免费源代码重新构建而成，外观与母发行版红帽企业级Linux如出一辙，使用YUM来管理软件包，在桌面端测试一下服务器的运作原理。

View details »

Fedora

Fedora是红帽公司的一个测试平台，产品在成为企业级发行版之前，在该平台上进行开发和测试。Fedora有庞大的用户论坛，软件库中还有为数不少的软件包。Fedora同样使用YUM来管理软件包。

View details »

Arch

Arch采用Pacman管理软件，旨在从开发者的角度而不是从用户的角度做到力求简单，Arch配置和安装起来超容易，它真是一款面向高手的发行版，让你可以了解Linux系统的每一个细枝末节。

View details »

OpenSuse

OpenSuse使用Yast来管理软件包，随带snapper快照管理工具，因而可以恢复或使用旧版的文件、更新和配置，SUSE在管理员当中的名气很大大，这得益于Yast以及其它能够自动管理任务的应用。

View details »

elementaryOS

Elementary OS是一款基于Ubuntu精心打磨美化的桌面Linux发行版，号称 “最美的Linux”，Elementary最早是Ubuntu的一个美化主题项目，现在成了独立的发行版，它拥有类似于Mac的主题风格。

View details »

Mint

Mint是一份基于Debian和Ubuntu的Linux发行版，其目标是提供一种更完整的即刻可用体验，Mint是对用户友好而功能强大的操作系统，为家庭用户和企业提供一个免费易用，舒适优雅的操作系统。

View details »

Puppy

Puppy是一个可以直接存放在U盘并启动的微型Linux操作系统，几乎能在所有的硬件上运行，易于操作，虽然它只有几十MB，但它自带了大量的扫描仪、打印机、数码相机驱动以及音乐编辑软件。

View details »

Jolicloud

Jolicloud是一款针对上网本设计的操作系统，使用Linux内核，虽然号称是一款 WEB 操作系统，同样，用户也可以自由的将数据存储在本地硬盘上，实际上，这是一个经过优化了的Ubuntu商业发行版。

View details »

FreeBSD

FreeBSD是一种类UNIX操作系统，是由经过BSD、386BSD和4.4BSD发展而来的Unix的一个重要分支，FreeBSD为不同架构的计算机系统提供了不同程度的支持，其代码被好多其他系统借鉴。

View details »

PCLinuxOS

PCLinuxOS是一个完全开放的linux版本，秉承“简单易用，安全无忧”的理念，逐步成长为一个稳定、成熟的版本。在没有大公司的帮助下，开发团队奇迹一般把这样一个平民版本推动到Linux世界的巅峰。

View details »

Deepin

Deepin是由武汉深之度科技有限公司开发的Linux发行版，专注于使用者对日常办公、学习、生活和娱乐的操作体验的极致，适合笔记本、桌面计算机和一体机。它包含了所有您需要的应用程序。

View details »

Oracle Linux

Oracle Linux是Oracle公司在2006年初发布第一个版本，以对Oracle软件和硬件支持较好见长，专门针对Oracle软件与硬件进行优化，最重要的是Oracle数据库跑在OEL上性能可以提升超过75%。

View details »

Vulnerable

这是一款很特别的发行版，Damn Vulnerable Linux恰如其名：该死的易受攻击的Linux，它有意捆绑了配置不当的、很容易被不法分子攻击的软件，它的目的在于借机训练Linux管理员。
