在线课堂

互联网时代，足不出户就可以享受高质量的教育，与其花钱参加形形色色的培训班，这里有来自全球各地的教授与专家。

Codecademy

Codecademy是全球全球学习编程最好的地方之一，它提供免费的在线交互式编程课堂，包括Ruby,SQL,Java,Git,HTML,Js等一系列课程,曾经在这里学习的人都会惊叹于它绝佳的体验、超高的效率与专业的内容。

View details »

Coursera

Coursera由美国斯坦福大学两名计算机科学教授创办，旨在同世界顶尖大学合作，提供免费的网络公开课程，合作院校包括加州理工大学、斯坦福大学、普林斯顿大学等美国名校，北大、南大、复旦等也在其中。

View details »

极客学院

国内最大IT职业在线教育平台，最权威的android开发在线学习平台。推出了中国第一个FlappyBird、Swift、Apple Watch等相关课程 , 几乎垄断了所有最新的技术和最热的应用课程的首发，内容十分全面且实用。

View details »

Udemy

在线学习的集市，全球有超过千万学生在这里学习从编程到瑜伽到摄影的所有课程，1万8千门课中的每门都由专家教师教授，学生们可以以自己的节奏、在自己的时间内、从任何设备上学习（既有付费也有免费课程）。

View details »

GitHub

世界最大开源社区，Github更像是一个无声的课堂，任何开发者都不会错过这个地方，谷歌推特都会将他们的开源项目放在这里，你也可以将它作为自己项目托管平台，你还可以在这里找到超过500本编程资料。

View details »

慕课网

中国版的慕课，视频教学、在线实验与问答社区相结合是其最大的特点，课程完全免费，涵盖前端、后端、移动开发、数据处理、图像处理，分为基础课程、实用案例、高级分享三大类型，适合不同阶段的人群学习。

View details »

CSDN学院

CSDN学院是CSDN最新推出的一个面向中国软件开发者和IT专业人员的全新技术教育服务平台，课程类型囊括移动开发、编程语言、软件研发、系统/网络、云计算/大数据等诸多领域。

View details »

网易云课堂

网易公司研发的一款在线教育平台服务，其中计算机专业体系分大一至大四的课程，云课堂拥有笔记功能，学习者添加笔记时会自动保存视频的当前时间点，回顾笔记时就可观看当时视频。

View details »

可汗学院

可汗学院是由萨尔曼·可汗创立的一家教育性非营利组织，旨在于利用网络影片进行免费授课，现有关于计算机、数学、金融、物理等科目的内容，每段课程影片长度约十分钟，受到社会的广泛好评。

View details »

SEE

Stanford Engineering Everywhere简称SEE，由斯坦福大学开设的免费在线课程，该SEE课程组合，包括斯坦福大学最受欢迎的计算机科学导论，深受广大斯坦福大学的本科生的喜爱。

View details »

MIT

如果你已经了解了基本知识，想进一步探索编程背后的理论， 你就可以利用MIT的免费课件网站，MIT拥有大量工程和计算机科学相关的课程，它提供了包括考试、作业、设计和分析算法等的免费资源。

View details »

edX

edX是由哈佛大学和麻省理工学院联合创建的免费在线课程项目，目的是建立世界顶尖高校联合的共享教育平台，提高教学质量，推广在线教育，在这里你会学到最前沿技术和理论，邂逅最令人尊敬的导师。

View details »

Udacity

Udacity所提供的课程包括数学、编程、计算机科学、设计等等，目标是“让所有人享受教育”，相比其他同类网站，Udacity更像是一所可以让人自由驰骋的大学，全球三大慕课网站之一。

View details »

学堂在线

学堂在线是由清华大学研发的中文MOOC面向全球提供在线课程。任何拥有上网条件的学生均可通过该平台，在网上学习，学堂在线涵盖计算机、经管创业、理学、工程、文学等多个领域。

View details »

Codeschool

相较于其他网站，Code School的编程教学更主张动手实践，通过视频、编程挑战和屏幕录影等方式，为来自全球的用户讲授各种编程课程，非常适合新手入门和高手进阶。

View details »

实验楼

国内首家以实验为核心的IT在线教育平台，让用户通过动手实践的方式学会IT技术，致力于为计算机相关专业学生提供在线实验平台。课程分为基础课，项目课，评估课。

View details »

51CTO

51CTO学院课程涵盖云计算/大数据/移动开发/游戏开发/系统/运维/考试认证/编程语言/Web开发/软件研发/数据库/网络技术/服务器存储/大学课程/嵌入式/企业信息化/Office/设计。

View details »

Treehouse

Treehouse通过视频、测试、勋章等方式进行网站及应用开发教学，课程类别涵盖Web开发、移动应用开发、设计（移动应用、网站等）及业务管理——help you change the world.

View details »

Lynda.com

Lynda.com是由Bruce Heavin和Lynda Weinman的夫妇于上世纪90年代初所创立，起初Heavin是想让自己可爱的作家妻子获得满足感，如今它已经成为深受全球用户所喜爱的网络教育平台。

View details »

Skillfeed

Skillfeed主要提供移动开发、Web开发和设计等方面的课程培训，其课程不仅有初学者和高手之分，还有时间长短之分，用户完全可以根据自己的实际情况进行学习安排，它的移动开发和3D动画教学颇得开发者青睐。

View details »

Codeavengers

Code Avengers提供趣味十足的交互式编程课程， 教你如何使用JavaScript, HTML 和 CSS编写游戏，应用程序和网站，所有的教程都是通俗易懂，并通过有趣的案例和练习来逐步提升难度，非常适合初学者学习。

View details »

腾讯课堂

腾讯推出的专业在线教育平台,聚合大量教育机构和名师,下设职业培训、公务员考试、托福雅思、考证考级、英语口语、中小学教育等众多在线学习课程，还可以在QQ客户端找到课程，通过QQ客户端直接进入课堂。

View details »

OpenLearning

澳大利亚社交型MOOC学习平台OpenLearning。澳大利亚政府第一门MOOC课程承接平台；马来西亚教育部官方指定MOOC学习平台。



View details »

mooc学院

MOOC学院是果壳网旗下的一个学习社区，收录了主流的三大课程提供商Coursera、Udacity、edX的部分课程，并将大部分课程的课程简介翻译成中文，学习的过程中可以和同学讨论课程问题，记录自己的上课笔记。

View details »

中国大学MOOC

中国大学MOOC，是最大的中文MOOC(慕课)平台，由“爱课程网”与“网易云课堂”合作推出，承接教育部国家精品开放课程任务，打造在线教育“中国梦”。
