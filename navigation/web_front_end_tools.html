图片压缩与网页设计

网页加载速度用户体验的重要部分，如果网页中包含大量的图片，往往会拖慢网页加载速度，这个时候我们就需要对图片进行压缩，并且在压缩的同时尽量的保持图片的质量。

这里的网页设计工具是指在设计网页或者开发页面时用得到的表单工具、栅格系统、UI组件等，它们可以大大加快网页开发速度。

TinyPNG

TinyPNG是一个PNG 图片压缩优化工具，使用智能有损压缩技术（降低图片中的颜色数量），在保证图片效果的同时，可以大大降低图片的大小。

imageoptimizer

持多种不同格式图片的的优化压缩，其中包括GIF、JPG、PNG，可以从本机上传图片或者输入图片的网址来进行优化，还能在线将图片转换成不同格式。

View details »

imagereducer

支持GIF、JPG和PNG格式的图片优化,直接从本机上传或粘贴图片地址，该工具会压缩并优化成多种不同的尺寸和质量供你选择。

图好快

图片压缩在线工具,可同时批量在线处理50-5000张图片，并提供专业的国内外网站图片云加速功能，批量压缩需要收费。

智图

智图为图片智能选择合适的图片格式,支持在线jpg图片压缩,png图片压缩,并支持ps改变图片品质，简洁方便的网站，值得推荐。

Web Resizer

Web Resizer无损图片压缩，支持GIF、JPG、PNG图片格式，同时还可以调整图片大小，并提供旋转、增加亮度、边框等等效果的设置。

麦客表单

非常实用的表单工具，追踪每天表单的反馈情况，曝光率最高的嵌入式表单，最适合移动终端上传播的二维码，自动为填写表单的用户创建联系人档案。
